# InsightForge

InsightForge is a Dockerized, multi-service application built to demonstrate
industry-grade backend architecture using Node.js, PostgreSQL, Prisma, and a
Python-based AI service.

## Tech Stack
- Node.js (Backend API)
- PostgreSQL (Database)
- Prisma ORM
- Python (AI Service)
- Docker & Docker Compose
